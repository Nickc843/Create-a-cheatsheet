Symantics in HTML

Why use symantics?

Symantic-Tags Name Description Possible examples (if any)
<nav> Navigation Used to define a block of navigation links, can be used inside header but also on its own menus, table of contents
<main> Main Used to encapsulate the dominant conent within a webpage
<footer> Footer Content at the bottom of the subject file contact info, copyright info, terms of use, site map, reference to top of page links
<section> Section Defines elements in a document introduction, news items, and contact info
<article> Article Holds content that makes sense on it's own articles, blogs, comments, magazines
<aside> Aside An additional element that helps enhance another element but isnt required in order to understand the main content bibliographes, endnotes, comments, editorial sidebars
<figure> Figure Used to encapsulate media Image, illustration, diagram, code snippet
<figcaption> Figure Caption Used inside figure to add a caption to the attached media
<audio> Audio Used to encapsulate audio
<video> Video Used to encapsulate video
<embed> Embed Self closing, SHOULD NOT BE USED, can embed any media content including videos, audio, and gifs from an external source. This is a deprecated tag.
